The first release of the Spring Archaius Application.

#### Basic Usage
```bash
archaius-docgen.bat --source-dir ./application1-location 
```

#### Windows
```bash
archaius-docgen.bat \
    --source-dir ./application1-location \
    --output-dir docs \
    --application-name "MyApplication"
```

#### Linux
```sh
archaius-docgen.sh \
    --source-dir ./application1-location \
    --output-dir docs \
    --application-name "MyApplication"
```
